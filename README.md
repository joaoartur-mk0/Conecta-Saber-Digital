# Conecta Saber Digital 🚀

> **Status:** Em fase de planejamento e estruturação inicial.

## 📌 Sobre o Projeto
O **Conecta Saber Digital** é um projeto de extensão voltado para a alfabetização e cidadania digital da comunidade local. A infraestrutura e a documentação técnica (Playbooks) são gerenciadas nos bastidores pela equipe executora utilizando a cultura **Docs-as-Code**, metodologias ágeis e automação DevOps.

## 🏗️ Estrutura do Repositório
A organização das pastas segue os padrões de mercado para projetos que utilizam geradores de sites estáticos e CI/CD:

    /
    ├── .github/workflows/       # Esteiras de automação CI/CD (GitHub Actions)
    ├── design/                  # Ativos de UI/UX, Brandbook, Paleta de Cores e Tipografia
    ├── infra/                   # Arquivos Docker, configurações Nginx e scripts de deploy
    ├── docs/                    # Documentação interna da equipe (Não é o material do público)
    └── site/                    # Código-fonte do GoHugo e Playbooks (Markdown)

## 💻 Stack Tecnológica
*   **Site e Documentação:** GoHugo (Gerador de Site Estático) + Markdown
*   **Automação (CI/CD):** GitHub Actions
*   **Hospedagem:** VPS Oracle Cloud (Always Free Tier) + Nginx/Docker
*   **Gestão Ágil:** OpenProject (Self-hosted)
